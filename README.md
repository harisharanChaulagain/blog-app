# Blog Management Frontend Project

This is a frontend project for managing blogs. It allows users to login, create, view, edit, and delete blog posts. The project is built using **Next.js**, **Tailwind CSS**, **Redux Toolkit (RTK)** for state management, and **RTK Query** for API requests.

## Features

- User authentication with token-based login
- Create, edit, view, and delete blog posts
- Filter posts by category, search, and tags
- Pagination for posts
- Responsive design (desktop & mobile)
- Tailwind CSS for styling
- State management with Redux Toolkit
- API requests handled using RTK Query
- Toast notifications using Sonner

## Tech Stack

- **Frontend:** Next.js, React, TypeScript  
- **State Management:** Redux Toolkit (RTK)  
- **API Handling:** RTK Query  
- **Styling:** Tailwind CSS  
- **Mock Backend:** JSON Server  
- **Notifications:** Sonner Toast
